Changelog
=========

``gdscheck`` is pre-1.0 and under active development (currently ``0.1.0``); there are no
tagged releases yet, so this page is not yet populated with dated entries. Once the
project starts cutting releases, each one will get a section here summarising
user-visible changes (new checks, new PDKs, breaking deck/CLI changes, and notable
correctness fixes).

In the meantime, the commit history is the authoritative record of what changed and why.
